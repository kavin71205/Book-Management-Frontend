# Book Management System – Frontend

This is the **frontend** of the Book Management System built using **React**.  
It allows users to sign up, log in, and manage books by interacting with a secure backend API.

## Live Demo Link

- **Frontend URL (Vercel)**

- **Backend API**  
https://book-management-backend-rk86.onrender.com

---

## Features

- User Signup & Login
- JWT-based Authentication
- View Books
- Add New Books
- Update & Delete Books
- Logout functionality
- Responsive UI

---

## Tech Stack

- React
- JavaScript
- HTML
- CSS / Tailwind CSS
- Fetch API
